# USES Codebase Search

Use this skill for repository-grounded questions about `/Users/songzuoqiang/Documents/agent/code`.

默认优先使用完整根目录索引库 `examples/agent_code_index.db`。只有在调用方明确要求只查 `uses_codes` 子目录时，才退回 `examples/uses_codes_index.db`。

## Preferred workflow

1. Prefer MCP tools from this plugin:
   - `answer_codebase`
   - `ask_codebase`
   - `assemble_evidence`
   - `query_codebase`
2. Use `answer_codebase` when the user is asking a direct question about business logic or implementation location.
3. Use `ask_codebase` or `assemble_evidence` when you need to inspect the supporting evidence before answering.
4. Only fall back to the local HTTP API if the MCP server is unavailable:
   - `POST http://127.0.0.1:8000/answer`
   - `POST http://127.0.0.1:8000/ask`

## Response expectations

- Prefer returning the grounded final answer from `answer_codebase`.
- Cite `procedure_name`, `file_path`, and `line_start` / `line_end`.
- If the returned `answer_source` is `draft`, say it is grounded in indexed evidence and not generated by an external model.
- If the question is broad, summarize the top evidence blocks before giving the conclusion.
